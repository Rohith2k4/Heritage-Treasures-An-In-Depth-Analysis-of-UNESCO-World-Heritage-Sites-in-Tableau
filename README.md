# Heritage Treasures: UNESCO World Heritage Analysis
## 📌 Project Overview 
This project presents an in-depth analytical study of UNESCO World Heritage Sites using Tableau. The analysis focuses on geographical distribution, category classification, and historical trends through interactive dashboards.
